# Aquilarium

**踏足大地，翱翔天际。**

Aquilarium 是一个关于虚构世界观企划 **Project Resurrection** 的开放设定百科。它记录了这个世界观下的地理、历史、人物、概念与组织，以 Wiki 词条的形式组织，通过超链接将相关概念编织成网。

与传统百科全书不同，Aquilarium 所记载的并非现实世界的信息，而是一个独立虚构世界的内在逻辑与知识体系。

## 浏览

本站内容以词条为基本单位，通过目录树、超链接、搜索框和交叉引用进行探索。页面底部列出相关词条，正文右侧有浮动目录。

## 内容

现有词条涵盖五大领域：地点（7 个国家和地区）、人物、事件、概念（工程·文化·自然）、组织和部门。

## 技术

由 [Quartz](https://quartz.jzhao.xyz/) 驱动，部署于 GitHub Pages。内容使用 Markdown 编写，与 Obsidian 知识库同步。
