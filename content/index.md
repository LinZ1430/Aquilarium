# 🌊 Aquilarium

欢迎来到我的数字花园。

这里是我整理、思考和沉淀知识的地方。内容会随着时间不断生长、修改、连接——就像一座水族馆，每一篇笔记都是其中游动的生命。

## 浏览方式

- 左侧 **目录树** 按文件夹浏览
- 点击笔记中的 **链接** 跳转关联内容
- 使用 **搜索** 查找特定主题
- 右侧 **关系图谱** 查看笔记之间的连接
- 每篇笔记底部有 **反向链接**，看看哪些笔记引用了当前页面
